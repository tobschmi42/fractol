# fractol
My implementation of the 42 fractol project.
## Description

The fractol project aims to be a first introduction into graphical programming. The program shows a window of either a Mandelbrot or Julia set, with ability to choose which Julia set, as well as the option to zoom into the visualizations.

## Instructions

To compile the program the 'make' command may be used. To clean up the created files the user may use 'make clean' to clean up the .d and .o files, or 'make fclean' to remove the binaries as well.

The function may be started with './fractol'
If the program is started without parameters (or any invalid input for that matter) options for the right usage may be displayed.
To get a visualization of the Mandelbrot set './fractol 0' may be used.
For the Julia set call: 'https://en.wikipedia.org/wiki/Double-precision_floating-point_format'
